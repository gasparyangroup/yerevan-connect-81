

# План: Исправление окна сбора заявок

## Изменения в `ContactModal.tsx`

### 1. Поля формы
- **"Имя / Компания"** вместо "Полное имя" — placeholder: "Иван Иванов / ООО Компания"
- **"Номер / Telegram"** вместо "Номер телефона" — placeholder: "+374 XX XXX XXX или @username"
- **Проект** — автоматически заполненное read-only поле, показывающее название проекта, на который кликнул пользователь

### 2. Отображение по центру
- Модальное окно уже центрировано (`fixed left-1/2 top-1/2 -translate-x-1/2 -translate-y-1/2`), но убедимся что `max-h-[90vh]` и `overflow-y-auto` работают корректно и все поля видны на любых экранах

### 3. Автозаполнение проекта
- Поле "Проект" будет показано как информационный блок (не редактируемое), заполняется из `project.title` переданного через props

## Технические детали

Файл для изменения: `src/components/ContactModal.tsx`

- Иконка `UserIcon` остается для поля "Имя / Компания"
- Иконка `Phone` заменяется на `MessageCircle` (или остается `Phone`) для "Номер / Telegram"
- Лейблы и placeholders обновляются
- Блок с карточкой проекта (image + title + location) остается как визуальное отображение выбранного проекта
- Добавим поле "Проект" в форму как disabled input с автозаполнением из `project.title`
